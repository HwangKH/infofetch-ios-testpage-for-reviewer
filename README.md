# InfoFetch Demo Page

This page was created specifically for App Store review purposes.

It contains **sample content generated with fictitious information** so that Apple reviewers can safely test the Safari extension included in the app without accessing any real services or private data.

The information displayed on this page is **not real and has been intentionally created for demonstration and testing purposes only**.

## Purpose

The goal of this page is to provide a simple environment where reviewers can experience how the extension works.

When the extension is activated on this page, it will:

- Detect the structured sample data on the page
- Parse the information
- Send the parsed data to the app
- Display the summarized information inside the app interface

This allows reviewers to verify the functionality of the extension without requiring login access to any external websites.

## Important Notice

All airport, flight, immigration, and quarantine information shown on this page is **fictional**.

The content exists solely to demonstrate how the extension organizes and processes web information.

No real-world services or organizations are involved.

## How to Test

1. Enable the Safari extension in Settings.
2. Open this demo page in Safari.
3. Activate the extension.
4. The extension will extract the sample information and send it to the app.

This demonstrates the core behavior of the extension in a safe and controlled environment.
